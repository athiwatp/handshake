# handshake

a peculiar chat app for strange times

## Makefile commands

| Command      | Description                               |
| ------------ | ----------------------------------------- |
| `make build` | Create a handshake docker image           |
| `make shell` | Run an interactive shell in the container |
| `make clean` | Delete all docker images and containers   |
